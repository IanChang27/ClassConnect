About Us

Class Connect is a website where teachers can post assignments and courses. Students can access the courses and assignments. Class Connect has three different tabs. First, we have the course page where one can view their current courses and view the topics covered in each course. Second, we have the assignments page where one can go see any pending assignments and view the assignment details, materials needed to complete it, along with the estimated time that is needed. Finally, we have a resources page where one can search for the resources needed for each course.
